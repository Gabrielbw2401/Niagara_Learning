Différence entre Niagara System et Niagara Emitter

Dans Unreal Engine, le système Niagara repose sur deux concepts fondamentaux : System et Emitter. Comprendre leur rôle est essentiel pour concevoir des effets visuels modulaires et puissants.

Schéma explicatif

  • System : Conteneur global qui orchestre un ou plusieurs Emitters.
  • Emitter : Génère et contrôle un type de particule (fumée, feu, débris, etc.).
  • Un System peut combiner plusieurs Emitters pour créer des effets complexes.

Tableau comparatif

Aspect Niagara System Niagara Emitter
Rôle Regroupe et orchestre plusieurs Emitters pour former un effet complet. Définit le comportement d’un type de particule (spawn, update, render).
Réutilisation Spécifique à un effet ou une séquence (ex : explosion, pluie, magie). Peut être réutilisé dans plusieurs Systems (ex : Emitter "Fumée" utilisé dans "Explosion" et "Cheminée").
Paramétrage Paramètres globaux, contrôle la synchronisation des Emitters. Paramètres propres à la génération et au comportement des particules.
Exemple System "Explosion" = Emitter "Débris" + Emitter "Fumée" + Emitter "Flash" Emitter "Fumée" = particules grises, fade-out, turbulence
